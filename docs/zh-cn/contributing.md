### 安装环境

`zpan` 基于[Go](https://golang.org/)进行开发.

依赖:

- `make`
- [Go 1.13+](https://golang.org/doc/install)

克隆源码:

```sh
$ git clone git@github.com:saltbo/zpan.git
```

安装构建依赖:

```sh
$ make dep
```

运行单元测试

```sh
$ make test
```

### 测试你的修改

您可以为更改创建分支，并尝试从源代码进行构建:

```sh
$ make build
```
